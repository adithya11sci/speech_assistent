🚀 HOW TO RUN
================================================================================

ONCE YOU'VE DOWNLOADED WAV2LIP MODEL:

1. Activate virtual environment (if not active):
   .\venv\Scripts\activate

2. Add source media:
   - Image: input\images\avatar.jpg
   - OR Video: input\videos\avatar.mp4

3. Run the system:
   cd src
   python main.py

4. Speak into your microphone!

💡 TIPS
================================================================================

1. Source Video Tips:
   - Use 5-10 seconds of video (will loop)

   - Person looking at camera
   - Resolution: 720p or higher

   - Expected latency: 1.5-2.5 seconds
   - Groq API is FAST (no local LLM overhead)

3. Troubleshooting:
   - If audio issues: Check PyAudio device selection
   - If video lag: Reduce WAV2LIP_FPS in config.py
   - If API errors: Check GROQ_API_KEY in config.py
