⚡ WHAT THIS SYSTEM DOES:

   You speak ──> AI hears (Whisper STT)
              ──> AI thinks (Llama 3.1)
              ──> AI speaks (Edge-TTS)
              ──> Avatar lip-syncs (Wav2Lip)
              ──> Display shows talking avatar!

   Target latency: 1.5-2.5 seconds (speech end → lips moving)

🔍 TROUBLESHOOTING:

   Problem: Setup script fails
   Solution: Read error message, check QUICKSTART.md for manual steps

   Problem: "CUDA not available"
   Solution: Install CUDA Toolkit, reinstall PyTorch with CUDA support

   Problem: "No module named 'something'"
   Solution: Activate venv, run: pip install -r requirements.txt

   Problem: "No face detected"
   Solution: Use clearer image with visible face, adjust threshold in config.py

   Problem: "CUDA out of memory"
   Solution: Close other GPU apps, reduce LLAMA_N_CTX in config.py

   Problem: Microphone not working
   Solution: Check Windows privacy settings, grant microphone permission

   → Full troubleshooting guide in QUICK_REFERENCE.md

🎨 CUSTOMIZATION:

   All settings in: src/config.py

   Key settings:
   - SOURCE_IMAGE or SOURCE_VIDEO - Your avatar
   - LLAMA_N_GPU_LAYERS - GPU usage (-1 = full GPU)
   - TOKEN_BUFFER_SIZE - Response speed
   - RMS_THRESHOLD - Microphone sensitivity
   - VOICE_NAME - TTS voice

   See README.md for complete configuration guide


⚡ PERFORMANCE TIPS:

   Faster responses:
   - Set LLAMA_N_GPU_LAYERS = -1 (use full GPU)
   - Set TOKEN_BUFFER_SIZE = 10 (lower = faster start)
   - Use WAV2LIP_USE_FP16 = True

   Lower VRAM:
   - Reduce LLAMA_N_CTX to 1024
   - Use smaller Whisper model (tiny.en)

   Better quality:
   - Increase LLAMA_N_CTX
   - Use larger Whisper model (base.en)
   - Disable WAV2LIP_USE_FP16
